## Leveling

Requirement: 

    Exp total surpasses current threshold

Result:

    1) Player class base stats increased
    2) Exp threshold increased

### Color Code

    Player: Lavender
    CPU: Pink

```plantuml

title Leveling

start
#pink:Exp threshold reached;
#pink:Player base stats updated;
#pink:Exp threshold increased;
stop
```
